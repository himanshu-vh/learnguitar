# Guitar Fretboard Visualizer

An interactive web application to visualize guitar scales and modes on the fretboard.

## Features

- 🎼 Support for 12 root notes and 9 scale types:
    - 'Major'
    - 'Minor'
    - 'Pentatonic'
    - 'Blues'
    - 'Dorian'
    - 'Mixolydian'
    - 'Lydian'
    - 'Phrygian'
    - 'Locrian'
